🗺️ NeuroNav (v1.0 Hackathon Edition)
NeuroNav is a full-stack, accessibility-focused web application built for the Youth Code x AI Hackathon. Inspired by the hackathon's mission to support the Akhil Autism Foundation, NeuroNav is designed to assist neurodiverse students—specifically those dealing with academic anxiety, ADHD, or executive dysfunction—by transforming overwhelming task descriptions into manageable, bite-sized micro-steps.

🚀 The Core Solution
When users look at a massive project deadline, executive dysfunction can cause mental paralysis. NeuroNav offloads that psychological burden:
The Focus Hub: Users input a daunting task and adjust an Overwhelm Slider (Levels 1–5) based on their current anxiety.
AI Task Decompression: A Next.js backend securely feeds this context into Google's Gemini API using a highly structured system prompt.
The Dopamine Loop: The app strips away the chaos, returning a Gentle Roadmap of tiny actions complete with contextual emojis, micro-time estimates, an interactive progress meter, and real-time coaching cheers.

🛠️ The Tech Stack
NeuroNav is built as a unified, full-stack application architecture:
Frontend: Next.js (React), Tailwind CSS, Lucide Icons
Backend: Next.js Serverless API Router (/api/breakdown)
AI Core Integration: Google Gemini API
Deployment Hosting: Vercel

💻 Local Installation & Setup
Follow these steps to run NeuroNav locally on your computer.
1. Clone the Repository
git clone https://github.com/Masngo/neuronav.git 
cd neuronav
2. Install Dependencies
npm install
3. Configure Environment Variables
Create a file named .env.local in the project root.
# Google Gemini API Key from Google AI Studio
AI_API_KEY=YOUR_GEMINI_API_KEY_HERE
Note: .env.local is included in .gitignore to ensure private API keys are never committed to the repository.
4. Start the Development Server
npm run dev
Open your browser and visit:
http://localhost:3000

📝 Devpost Submission Text
Inspiration
Academic anxiety and executive dysfunction are significant barriers for many students, particularly neurodiverse learners who may experience paralysis when faced with large assignments. Inspired by the Youth Code x AI Hackathon's mission to support the Akhil Autism Foundation, NeuroNav was created to transform overwhelming academic tasks into simple, stress-free, actionable steps that build confidence and reduce anxiety.

What it does
NeuroNav helps reduce mental overload by converting large project descriptions into personalized, bite-sized task checklists. Users describe a task and set an "Overwhelm Level" using a simple slider. The application uses generative AI to produce a Gentle Roadmap containing manageable micro-actions, estimated completion times, and supportive emojis. As users complete tasks, a progress tracker updates in real time while an AI Coach provides encouraging messages that reinforce motivation through positive feedback.

How we built it
NeuroNav is a full-stack web application built with Next.js and Tailwind CSS, providing a fast, responsive, and accessible user experience. A secure Next.js Serverless API route connects the frontend to Google's Gemini API, which processes user input using carefully engineered prompts to generate structured responses. The application is deployed on Vercel, with source code maintained across GitHub and GitLab repositories to support collaboration and redundancy.

Challenges we ran into
One of the biggest technical challenges involved ensuring reliable AI responses. We engineered the backend prompts to require Gemini to return valid, structured JSON rather than free-form text. This approach ensured consistent parsing by the backend and prevented rendering issues within the responsive frontend. Additional effort was devoted to maintaining layout stability, accessibility, and an intuitive user experience while integrating AI-generated content.
